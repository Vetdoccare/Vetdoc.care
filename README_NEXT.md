# VetDoc.Care — Next.js scaffold

This folder contains a minimal Next.js scaffold that serves the existing `index.html` content.

Quickstart:

```bash
npm install
npm run dev
```

Notes:
- The Next page `pages/index.js` extracts the original `index.html` styles at build time and injects them, preserving appearance.
- Interactive scripts were ported to React; further cleanup is optional.
